The tcsh mailing lists

There are three tcsh mailing lists:
tcsh@mx.gw.com
The tcsh maintainers and testers' mailing list.
tcsh-bugs@mx.gw.com
Open bug and user comment discussion.
You can subscribe to either of these lists by visiting http://mx.gw.com/

To file a bug report or a feature suggestion (preferably with code), please visit http://bugs.gw.com/
