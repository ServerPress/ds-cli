This is the GNU groff document formatting system. The version number is given in the file VERSION.

Included in this release are implementations of ‘troff’, ‘pic’, ‘eqn’, ‘tbl’, ‘grn’, ‘refer’, ‘−man’, ‘−mdoc’, ‘−mom’, and ‘−ms’ macros, and drivers for ‘PostScript’, ‘PDF’, ‘TeX DVI’ format, ‘HP LaserJet 4’ printers, ‘Canon CAPSL’ printers, ‘HTML’ and ‘XHTML’ formats (beta status), and typewriter-like devices. Also included is a modified version of the Berkeley ‘−me’ macros, the enhanced version ‘gxditview’ of the X11 ‘xditview’ previewer, and an implementation of the ‘−mm’ macros contributed by Jörgen Hägg.

See the file ‘INSTALL’ for installation instructions on how to install from a release. You require a C++ compiler.

The file ‘INSTALL.REPO’ contains supplementary instructions for building directly from a clone of the repository or a snapshot tarball.

The file ‘NEWS’ describes recent user-visible changes to ‘groff’.

‘groff’ is free software. See the file ‘COPYING’ for copying permission.

The file ‘PROBLEMS’ describes various problems that have been encountered in compiling, installing, and running ‘groff’.

The file ‘MORE.STUFF’ contains information about add-on packages useful for ‘groff’.
