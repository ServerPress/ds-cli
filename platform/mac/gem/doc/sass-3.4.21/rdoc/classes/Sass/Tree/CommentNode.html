Class Sass::Tree::CommentNode
In: lib/sass/tree/comment_node.rb

A static node representing a Sass comment (silent or loud).

@see Sass::Tree

Methods

==   invisible?   lines   new  

Attributes

resolved_value  [RW]  The text of the comment after any interpolated SassScript has been resolved. Only set once \{Tree::Visitors::Perform} has been run.

@return [String]

type  [RW]  The type of the comment. `:silent` means it‘s never output to CSS, `:normal` means it‘s output in every compile mode except `:compressed`, and `:loud` means it‘s output even in `:compressed`.

@return [Symbol]

value  [RW]  The text of the comment, not including `/*` and `*/`. Interspersed with {Sass::Script::Tree::Node}s representing `#{}`-interpolation if this is a loud comment.

@return [Array<String, Sass::Script::Tree::Node>]

invisible?()

Returns `true` if this is a silent comment or the current style doesn‘t render comments.

Comments starting with ! are never invisible (and the ! is removed from the output.)

@return [Boolean]
