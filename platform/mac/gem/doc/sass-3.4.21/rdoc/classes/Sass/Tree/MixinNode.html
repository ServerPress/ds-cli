Class Sass::Tree::MixinNode
In: lib/sass/tree/mixin_node.rb
Parent: Node

A static node representing a mixin include. When in a static tree, the sole purpose is to wrap exceptions to add the mixin to the backtrace.

@see Sass::Tree
